# 🌊 SHORE - Educational Platform

Welcome to the SHORE Educational Platform! This is a modern, full-stack application designed to manage students, volunteers, attendance, announcements, and recitations with a clean and interactive interface.

## 🔗 Live Links
- **GitHub Repository:** [https://github.com/AlanSerios/SHORE.edu](https://github.com/AlanSerios/SHORE.edu)
- **Backend API (Render):** [https://shore-backend.onrender.com](https://shore-backend.onrender.com)
- **Frontend App (Vercel):** *(Add your Vercel URL here once deployed)*

## 🚀 Features
- **Role-Based Dashboards:** Separate views for Admins, Volunteers, and Students.
- **Attendance Tracking:** Easily log and monitor student and volunteer attendance.
- **Announcements:** Post real-time announcements for all users.
- **Recitation & Leaderboards:** Award points for recitations and track scores on a live leaderboard.
- **PDF Report Generation:** Generate detailed progress reports for students instantly.

## 🛠️ Tech Stack
- **Frontend:** React + Vite (HTML/JS/CSS)
- **Backend:** Python + Flask
- **Data Storage:** JSON Files
- **Deployment:** Vercel (Frontend) & Render (Backend)

## 💻 Local Development

### Prerequisites
- Node.js installed
- Python installed

### Running the App Locally
You can start the entire app with a single click using the included batch script:
1. Double click on `start.bat`
2. It will automatically create a virtual environment, install Python dependencies, and start the backend server.
3. Open a separate terminal, navigate to the `frontend` folder, and run:
   ```bash
   npm install
   npm run dev
   ```

Enjoy exploring SHORE!
